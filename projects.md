# kleopatra landing

**Type**: Business landing page (cosmetics studio)
**Framework**: Next.js 14 · App Router · TypeScript · Tailwind CSS
**Language**: German (de-AT)
**Client**: Kosmetiksalon Kleopatra, 1010 Wien

## Description

A modern Next.js landing site replacing the old Avada/Fusion Builder WordPress installation for Kosmetiksalon Kleopatra — a cosmetics studio for women and men in Vienna's 1st district.

## Key Pages

| Route | Description |
|---|---|
| `/` | Home — hero image + intro section (services, address, hours, contact) |
| `/behandlungen` | Treatments overview |
| `/kontakt` | Contact page |
| `/artikel` | Blog / articles |

## Status

🚧 Active migration from WordPress → Next.js.
SEO preservation is the top priority — see `agents.md` for rules.
